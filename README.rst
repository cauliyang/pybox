Aims
--------

- **Simple**:
  - A simple and easy to use Python library for many annoy task.


Features
------------

- `Google Driver`_:
  - A simple and easy to download files by sharing link of Google Driver.

- More to come...

Installation
------------

You can install *Pybox* via pip_ from PyPI_:

.. code:: console

   $ pip install pybox
